# 视频瘦身器 (Video Slimmer)

基于 Flutter 开发的 iOS 视频压缩应用，功能简洁强大。

## 运行
```bash
flutter pub get
flutter run
```

## 开发环境
- Flutter 3.27.4 (稳定版)
- Dart 3.6.2
- Xcode 13.0+ (推荐 14.0+)
- CocoaPods 1.10+

## 系统要求
- 最低支持：iOS 17.0+
