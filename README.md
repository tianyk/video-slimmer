# 视频瘦身器 (Video Slimmer)

基于 Flutter 开发的 iOS 视频压缩应用，功能简洁强大。

## 运行
```bash
flutter pub get
flutter run --device-id "iPhone"
```

## 技术栈
- Flutter 3.27.4 + Dart 3.6.2
- flutter_bloc + flutter_riverpod 状态管理
- ffmpeg_kit_flutter_new 视频处理
- photo_manager 相册访问
- go_router 路由管理

## 开发环境
- Flutter 3.27.4 (稳定版)
- Dart 3.6.2
- Xcode 13.0+ (推荐 14.0+)
- CocoaPods 1.10+

## 系统要求
- 最低支持：iOS 17.0+
